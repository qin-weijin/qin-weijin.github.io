## 操作

**指定伪元素"pseudoElt"，返回"element"的 CSSStyleDeclaration 对象，只读**

	window.getComputedStyle(element,"pseudoElt")

**获取指定元素 style 内联样式，读写**

	element.style

**指定属性名 propertyname 返回它的值**

	CSSStyleDeclaration.getPropertyValue(propertyname)

**通过更换伪元素类控制样式变化**

	.red::before { content: "red"; color: red; }
	.green::before { content: "green"; color: green; }

**使用 js 修改伪类**

	// 创建 style 加入 head 获取 stylesheet

1.在 <head> 中创建 style 

	var style = document.createElement("style"); 
	document.head.appendChild(style); 
	sheet = style.sheet; 

2.插入样式
- stylesheet: 样式表对象
- rule: 样式表 str 写法
- index: 索引 

	sheet.insertRule('.red::before { color: green }', 0)
	stylesheet.insertRule(rule, index)

**修改 data-attr 私有属性控制伪类**

	<div class="red" data-attr="red"></div>
	<style>
		.red::before {
			content: attr(data-attr);
			color: red;
		}
	</style>
	<script>$('.red').attr('data-attr', 'green');</script>

### 避免高消耗属性

- box-shadows
- border-radius
- transparency
- transforms
- CSS filters（性能杀手）
- Float

### Display 规范

规范使用 Display 它会影响页面渲染

- `inline` - 后不应该再使用 width、height、margin、padding 以及 float
- `inline-block` - 后不应该再使用 float
- `block` - 后不应该再使用 vertical-align
- `table-*` - 后不应该再使用 margin 或者 float

### 动画优化

- JavaScript 动画尽量使用 requestAnimationFrame，避免 setTimeout, setInterval。
- 尽量使用 CSS 声明动画
- 使用 absolute 取代 translate 

### 选择器优化

- 预览器会从右到左遍历 CSS 选择器给定范围
- 最右端 CSS 选择器为"关键选择器"
- 避免使用通用选择器
- 避免使用多层标签选择器

**避免使用标签限制 Class**

	/* Not recommended */
	treecell.indented {…}

	/* Recommended */
	.treecell-indented {…}

**避免使用子选择器**

	/* Not recommended */
	treehead treerow treecell {…}

	/* Recommended */
	treehead > treerow > treecell {…}

**使用继承减少子选择器**

	/* Not recommended */
	`#bookmarkMenuItem > .menu-left { list-style-image: url(blah) }`

	/* Recommended */
	`#bookmarkMenuItem { list-style-image: url(blah) }`

# CSS 媒体查询

	@media <mediatype> <description> (feature){ /* Some Css */ }

mediatype - 设备类型

- `all` - 所有
- `print` - 打印
- `screen` - 屏幕
- `speech` - 屏幕预览器发声工具，视障
- `aural` - 语音和音频合成器
- `braille` - 盲人用点字法触觉回馈设备
- `embossed` - 分页的盲人用点字法打印机
- `handheld` - 小的手持的设备
- `projection` - 方案展示，比如幻灯片
- `tty` - 使用固定密度字母栅格的媒介，比如电传打字机和终端
- `tv` - 电视机类型的设备

description - 描述

- `and` - 连接多个设备
- `not` - 应用于除该设备外所有设备
- `only` - 只应用于

feature - 媒体功能

- `width`
- `min-width` - 可见区最小宽度
- `max-width` - 最大宽度
- `device-width` - 宽度
