# 全局属性

全局属性是所有 HTML 元素共有的属性，包括：

- HTML 基本属性
- 事件处理程序属性
- `aria-*` - 是一组用于支持残障人士访问的属性。
- `xml: lang`、`xml:base` - XHTML 兼容属性。
